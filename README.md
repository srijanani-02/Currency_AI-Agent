# 💱 Currency Converter AI Agent

An AI-powered Currency Converter built using **Python**, **Streamlit**, **Frankfurter API**, and **OpenRouter**. The application provides real-time currency conversion and AI-powered conversational responses.

## 🚀 Live Demo

🔗 **Live Demo:** https://currencyai-agent-eynwyfzhnwo23ux2swjk8e.streamlit.app/

## 📌 Features

- 💱 Convert currencies using real-time exchange rates
- 🤖 AI-powered general chat using OpenRouter
- 🧠 Intelligent tool calling for currency conversion
- 💾 Conversation memory support
- 🔒 Secure API key management
- ☁️ Deployed on Streamlit Community Cloud

## 🛠️ Tech Stack

- Python
- Streamlit
- Frankfurter API
- OpenRouter
- Requests
- Python-dotenv
- Git & GitHub

## ⚙️ How It Works

1. User enters a currency conversion query in the Streamlit application.
2. The AI agent analyzes the user's request.
3. Currency conversion requests are processed using the Frankfurter API.
4. General conversations are handled using OpenRouter.
5. The final response is displayed in the Streamlit interface.
